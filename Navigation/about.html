Sobre o projeto


Este website é um projeto criado para a cadeira de Interfaces Web da Universidade do Minho do mestrado de Humanidades Digitais.

O nosso objetivo é apresentar uma experiência online interativa para os visitantes conhecerem e explorarem a vida e obra de Camilo Castelo Branco.
